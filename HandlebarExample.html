PreCompilation in Handlebar JS

handlebars-runtime.js

This is the runtime handlebar js file. It's smaller than the full handlebars script because it doesn't need to worry about compiling templates. It assumes you have pre-compiled them.

flight.js

This is the pre compiled template file, compiled at the backend to avoid the compilation overhead at client side.
